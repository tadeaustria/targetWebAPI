Overview

This tool generates shooting target including shot information with given parameters.

Try using this basic example picture page

Parameters

x1, y1, x2, y2 ...
Coordinate pair values when numbers correspond. Coordinate values are with respect to the middle of the target. Can also be negative integer values. If both values are positive the shot is buttom right. It is recommended to use shot informations in multiples of 5 as this meets the table design. Example: Shot bottom right
withtable
If this argument is set to None the table and the caculated center are discarded. Default is True. Example: No Table
withtenth
If this argument is set to any value the table contains tenths. This argument has only a effect if withtable is true. Default is False. Example: With tenth
type
This has influence on the type of target and its color. Following values can be used b - Default black; r - red; g - green. Here r has a special function, that the ring table is replaced by the best Teiler table. Default is b. Examples: Green target Red target
